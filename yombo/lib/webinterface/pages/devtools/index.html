Developer Tools

Developer tools allows you to create new types of automation items. This includes new commands, device types, and modules that are available. Settings here do no directly affect the local gateway.

Public VS Local (private)Public items are available to every Yombo user and gateway. Local items are only available to your user or gateway.

For items that affect the local gateway, see the settings area.

Create or manage automation items. See help (above) for details.

  • Commands - Used to control devices.
    • Index - List commands available.
    • Add - Add a new commands.
  • Device Types - Types of devices available.
    • Index - List device types available.
    • Add - Add a new device type.
  • Input Types - Types of input available.
    • Index - List input types available.
    • Add - Add a new input type.
  • Modules - Modules users can download and use on their gateways.
    • Index - List modules available.
    • Add - Add a new module.
